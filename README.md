# LP画像の配置方法

## 画像ファイルの配置手順

1. **Windowsエクスプローラーで開く**
   - `\\wsl$\Ubuntu\home\user\claude-test\sample-lp\images`

2. **画像ファイルをコピー**
   - `hero-bg.jpg` - ヒーロー背景画像
   - `logo.png` - ロゴ画像
   - `product.png` - 商品画像
   - など、必要な画像をこのフォルダに入れてください

3. **ファイル名を教えてください**
   - 例：「hero-bg.jpgに背景画像、logo.pngにロゴを入れました」

画像を配置したら、HTMLで自動的に参照して実装します！
